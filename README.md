# yamasusu

当サイトへのリンクは[こちら](https://yamasusu.herokuapp.com)  
  
## Description  
yamasusuは、登山の魅力を伝えるサイトである。  
PC スマホ共に閲覧可能なレスポンシブデザインを取り入れた。  
タスクランナー(gulp),sass記法,CSS設計(FLOCSS)を使用することで、  
実務に役立てるスキルを学ぶことができた。  

## yamasusuの主な機能  

1.ハンバーガーメニュー  
2.画像スライダー
